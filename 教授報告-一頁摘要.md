# 自適應教學系統 - 一頁摘要（給教授報告用）

## 🎯 核心概念（30秒說明）

**類比心理診斷 AI**：
```
心理 AI：憂鬱程度(輕/中/重) → 調整問診方式
訓練 AI：理解程度(優/良/普/差) → 調整教學模式
```

**4種自動切換的教學模式**：
1. **挑戰模式** (平均≥8.5分)：複雜問題，無提示
2. **標準模式** (平均6.5-8.5)：適中難度，適度引導
3. **鷹架模式** (平均4.0-6.5)：分解步驟，頻繁提示
4. **補救模式** (平均<4.0)：基礎概念，大量提示

---

## 📊 完整範例（1分鐘說明）

### 學員學習「熱膨脹」的完整軌跡

| 次數 | 得分 | 平均 | 系統判斷 | 系統反應 |
|------|------|------|----------|----------|
| 1-2次 | 6,5分 | 5.5 | ⚠️ 表現不穩 | 切換至**鷹架模式**：分解步驟引導 |
| 3-5次 | 7,8,9分 | 7.0 | ✅ 持續進步 | 恢復**標準模式**：正常難度 |
| 6-7次 | 9,9分 | 8.8 | 🎯 連續高分 | 切換至**挑戰模式**：複雜情境，無提示 |

**關鍵**：同樣問「熱膨脹」，第2次時系統會手把手教，第7次時系統會丟複雜的多參數異常讓學員獨立分析。

## 💡 技術亮點（3個重點）

### 1. 多維度評估（不只看單次得分）
- 單次得分 (30%)
- **最近5次平均** (40%) ← 核心指標
- 主題掌握度 (20%)
- 學習趨勢 (10%)

### 2. 主題追蹤（自動識別弱點）
```
thermal (熱學)：8.5分 → ✅ 已掌握
vacuum (真空)：4.2分 → ⚠️ 需加強
optical (光學)：7.8分 → ✅ 精通中
```

### 3. 學習分析報告（數據驅動）
```json
{

  "weak_topics": ["vacuum"],
  "trend": "improving",
  "suggested_actions": ["💡 vacuum 需額外練習"]
}
```

## ❓ 應對教授提問（快速參考）

**Q: 跟一般LMS有什麼不同？**
A: 即時調整（不是事後分析）+ 多維度評估（不只看單次得分）

**Q: 如果學員亂答呢？**
A: 用最近5次平均，避免單次異常；會標記可疑回答

**Q: 有理論依據嗎？**
A: Scaffolding、Cognitive Load、ZPD 三大理論 + 參考您的心理AI例子

**Q: 需要強大硬體嗎？**
A: 不需要！模式切換是數學計算（毫秒級），只有生成內容用LLM

**Q: 有實驗數據嗎？**
A: 目前實作完成，計劃進行對比實驗（固定難度 vs 自適應）
