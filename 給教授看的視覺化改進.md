## 🎯 主要改進

### 1. 真實設備外觀
- ✅ 採用半導體蝕刻設備（Etching Chamber）設計
- ✅ 金屬質感外殼 + 透明真空腔體
- ✅ 專業工業設備風格
- ✅ 清楚標示各部件名稱

### 2. 異常部位發光效果 🔥
完全符合您的要求！

**正常狀態（綠色）**
```
✓ 冷卻水系統：藍綠色，流動動畫
✓ 真空腔體：綠色柔和發光
✓ 溫度感測器：綠色正常顯示
✓ 過濾網：灰色正常
```

**異常狀態（發光警示）**
```
⚠ 冷卻流量低 → 橙色脈動 + 流動減慢
🔥 溫度過高 → 紅色快速閃爍
⚡ 真空洩漏 → 紅色強烈發光
⛔ 過濾網堵塞 → 橙色脈動
```

### 3. 浮動標籤清楚標示
- 每個異常部位都有**浮動標籤**
- 黑底白字，清楚易讀
- 顏色邊框區分嚴重程度
- 標示具體問題（如：「冷卻流量過低」）

### 4. 動態流動效果
- **冷卻水管**：藍色流動動畫
- 異常時：流速變慢 + 顏色變紅
- 正常時：平穩流動 + 藍色
