## 🚀 如何啟用互動式視覺化

### 方法 1：自動切換（推薦）

```bash
cd semiconductor_training_system
python 切換視覺化模式.py
```

選擇 **模式 4**（互動式視覺化）

### 方法 2：手動修改

編輯 `interface/simulation_interface.py`：

```python
# 找到這一行（約第 21 行）
from interface.equipment_visualizer_photo import PhotoEquipmentVisualizer

# 改成
from interface.equipment_visualizer_interactive import InteractiveEquipmentVisualizer

# 找到這一行（約第 50 行）
self.equipment_visualizer = PhotoEquipmentVisualizer(...)

# 改成
self.equipment_visualizer = InteractiveEquipmentVisualizer()
```

## 📸 準備部件特寫照片（選用）

系統已內建 SVG 示意圖，但如果你有真實的部件照片，效果會更好！

### 部件照片資料夾結構：

```
interface/images/components/
├── light_source.jpg      # 曝光光源特寫
├── projection_lens.jpg   # 投影鏡頭特寫
├── vacuum_chamber.jpg    # 真空腔體特寫
├── wafer_stage.jpg       # 晶圓載台特寫
├── cooling_system.jpg    # 冷卻系統特寫（重要！）
├── alignment_system.jpg  # 對準系統特寫
└── air_filter.jpg        # 空氣過濾特寫
```

### 建立資料夾：

```bash
mkdir "interface\images\components"
```

### 照片建議：

1. **冷卻系統**（最重要，因為常故障）
   - 拍攝管路、閥門、過濾網
   - 可以看到溫度感測器
   - 解析度建議：1920x1080

2. **投影鏡頭組**
   - 鏡頭特寫
   - 可以看到鏡片
   - 解析度建議：1920x1080

3. **真空腔體**
   - 腔體內部或外觀
   - 密封圈、O環
   - 解析度建議：1920x1080

### 如果沒有照片？

**沒關係！** 系統已內建高品質 SVG 示意圖：
- 曝光光源：紫色發光示意圖
- 投影鏡頭：玻璃鏡片示意圖
- 真空腔體：半透明腔體示意圖
- 冷卻系統：藍色水管示意圖
- 等等...

## 🔧 支援的互動功能

### 1. 部件點擊

**所有部件都可點擊**：
- 曝光光源
- 投影鏡頭組
- 真空腔體
- 晶圓載台
- 冷卻系統 ⭐（故障時紅燈）
- 對準系統
- 空氣過濾

### 2. 狀態指示

**三種顏色**：
- 🟢 綠色 = 正常
- 🟠 橙色 = 警告
- 🔴 紅色 = 異常（閃爍）

### 3. 詳細資訊

**每個部件都有**：
- 功能說明
- 關鍵參數（當前值）
- 常見問題
- 當前狀態診斷
- 建議動作

### 4. 返回總覽

點擊左上角 **「← 返回總覽」** 按鈕

## 💡 使用場景

### 場景 1：新人訓練

**學員**：「冷卻系統在哪裡？」
**系統**：總覽圖上，右側標示「冷卻系統」，點擊它
**學員**：*點擊*
**系統**：顯示冷卻系統近距離照片，標示各部件位置

### 場景 2：故障診斷

**系統**：偵測到冷卻流量過低，總覽圖上冷卻系統**亮紅燈閃爍**
**學員**：點擊紅燈部位
**系統**：
1. 顯示冷卻系統特寫照片
2. 紅色邊框標示故障位置
3. 顯示當前參數：流量 3.5 L/min（過低）
4. 列出常見原因：過濾網堵塞、管路洩漏
5. 建議動作：檢查過濾網

### 場景 3：設備學習

**學員**：「投影鏡頭長什麼樣？」
**系統**：總覽圖點擊「投影鏡頭組」
**學員**：*點擊*
**系統**：
1. 顯示鏡頭組特寫照片/示意圖
2. 說明：4倍縮小投影光學系統
3. 關鍵參數：40+ 片石英鏡片、NA 1.35
4. 常見問題：鏡片污染、光學偏移

class InteractiveEquipmentVisualizer:

        if selected_component == "cooling_system":
            # 顯示冷卻系統特寫
            return self._generate_component_detail(...)
