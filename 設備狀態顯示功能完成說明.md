## 新增功能

### 設備狀態檢視區塊

在介面右側「參數監控」下方新增了「設備檢查」區塊，會顯示你檢查過的設備的**實體狀態**。

## 使用方式

### 1. 檢查設備

輸入檢查指令，例如：
- 「檢查冷卻水」
- 「檢查真空」
- 「檢查過濾網」
- 「檢查溫度」

### 2. 查看實體狀態

檢查後，**設備檢查區塊**會顯示詳細的實體資訊：

#### 真空系統範例（正常）
```
✓ 真空系統
  壓力：1.00e-06 Torr（正常）
  真空泵：運作正常
  管路：密封良好
```

#### 真空系統範例（異常）
```
⚠ 真空系統
  壓力：5.00e-05 Torr（異常）
  真空管：發現微小裂痕（O環老化）
  閥門：密封圈磨損
  建議：更換真空管O環密封圈
```

#### 冷卻水系統範例（異常）
```
⚠ 冷卻水系統
  流量：3.50 L/min（偏低）
  過濾網：堵塞（發現大量雜質）
  管路：O環可能老化
  建議：停機更換過濾網
```

#### 過濾網範例（堵塞）
```
⚠ 過濾網
  狀態：堵塞
  積累雜質：嚴重
  流通性：受阻
  建議：立即更換
```

## 完整使用流程示範

### 場景：冷卻系統故障

1. **開始情境**
   - 點擊「開始新情境」
   - 警報：冷卻系統異常！

2. **檢查參數**（輸入：檢查冷卻水）
   - 「參數監控」顯示：流量 3.5 L/min
   - **「設備檢查」顯示**：
     ```
     ⚠ 冷卻水系統
       流量：3.50 L/min（偏低）
       過濾網：堵塞（發現大量雜質）
       管路：O環可能老化
       建議：停機更換過濾網
     ```

3. **詢問學長**（輸入：流量這麼低，是過濾網的問題嗎？）
   - 對話歷史顯示學長回應
   - 設備檢查區塊保持不變，可以回看

4. **檢查過濾網**（輸入：檢查過濾網）
   - **「設備檢查」更新**：
     ```
     ⚠ 冷卻水系統
       流量：3.50 L/min（偏低）
       過濾網：堵塞（發現大量雜質）
       管路：O環可能老化
       建議：停機更換過濾網

     ⚠ 過濾網
       狀態：堵塞
       積累雜質：嚴重
       流通性：受阻
       建議：立即更換
     ```

5. **討論處理方案**（輸入：學長，我看過濾網堵得很嚴重，要怎麼處理？）
   - 學長會根據你觀察到的實體狀態給建議

## 支援的設備類型

以下設備檢查時都會顯示**實體狀態**：

| 設備 | 檢查指令範例 | 顯示資訊 |
|------|------------|---------|
| 冷卻水系統 | 檢查冷卻水 | 流量、過濾網狀態、管路O環 |
| 真空系統 | 檢查真空 | 壓力、真空管裂痕、閥門密封圈 |
| 溫控系統 | 檢查溫度 | 鏡頭溫度、散熱風扇、可能原因 |
| 過濾網 | 檢查過濾網 | 堵塞狀態、雜質累積、流通性 |
| 光源系統 | 檢查光源 | 強度、光學鏡片污染 |
| 電源系統 | 檢查電源 | 供電、電壓、插頭連接 |

## 技術實現

### 修改的檔案
- `interface/simulation_interface.py`

### 主要變更

1. **新增設備狀態追蹤**
   ```python
   self.equipment_status = {}  # 追蹤檢查過的設備
   ```

2. **新增設備狀態更新方法**
   - `_update_equipment_status()`: 根據檢查結果更新實體狀態
   - `_generate_equipment_status()`: 生成 HTML 顯示

3. **修改方法簽名**
   - `start_new_scenario()`: 返回值增加 `equipment_status_html`
   - `process_user_input()`: 增加 `equipment_status_html` 參數和返回值
   - `auto_progress()`: 增加 `equipment_status_html` 參數和返回值

4. **介面佈局更新**
   - 新增 `equipment_status_display` 組件（gr.HTML）
   - 位置：右側欄，參數監控下方

5. **事件綁定更新**
   - 所有事件（start, submit, timer）都包含 `equipment_status_display`
