# 📚 專案總覽

## 🎯 一分鐘快速理解

這是一個**半導體新人訓練系統**，讓新手可以在**虛擬環境**中練習處理設備故障，就像飛行員用模擬器訓練一樣。

### 核心價值

```
❌ 傳統方式              ✅ 本系統
   新人操作真機             虛擬設備練習
   出錯 → 設備損壞          出錯 → 即時指導
   訓練 3-6 個月           訓練 1-2 個月
   良率損失                零風險
```

---

## 🏗️ 系統架構（簡化版）

```
學員操作介面 (Gradio)
        ↓
   AI 三專家協助
   ├─ 診斷專家：告訴你哪裡壞了
   ├─ 操作專家：教你怎麼修
   └─ 安全專家：提醒你注意安全
        ↓
   虛擬曝光機
   (基於真實資料模擬)
```

## 🚦 快速開始（3 步驟）

### 步驟 1: 安裝套件

```bash
pip install -r requirements.txt
```

### 步驟 2: 準備資料

下載 [uci-secom.csv](https://www.kaggle.com/datasets/paresh2047/uci-secom) 並放到專案根目錄

### 步驟 3: 啟動系統

```bash
python run_training_system.py
```

然後開啟瀏覽器訪問 http://localhost:7860

## 🎮 使用流程

### 1️⃣ 選擇訓練難度
- EASY：對準漂移（簡單）
- MEDIUM：溫度/光強度問題（中等）
- HARD：真空洩漏/電氣故障（困難）

### 2️⃣ 開始訓練
系統隨機給你一個故障情境，例如：
> "你注意到真空計讀數異常上升，警報器開始響起..."

### 3️⃣ 診斷問題
- 查看 590 個感測器資料
- 請求 AI 專家協助
- 提出你的診斷

### 4️⃣ 執行操作
- 按照專家建議的步驟操作
- 系統記錄你的每個動作
- 提供即時反饋

### 5️⃣ 取得評分
- 診斷準確性 (30%)
- 操作正確性 (40%)
- 安全合規性 (30%)
- 時間效率 (加分)

## 🛠️ 常見問題

### Q1: 需要什麼背景知識？
**A:**
- 基礎 Python（會寫 if/for 就夠）
- 了解半導體製程（知道曝光機是什麼）
- 不需要機器學習背景

### Q2: 資料集從哪裡來？
**A:**
- UCI Machine Learning Repository
- Kaggle: "UCI SECOM Dataset"
- 搜尋 "uci-secom.csv"

### Q3: 可以客製化嗎？
**A:**
可以！修改這些檔案：
- `scenario_generator.py`: 新增情境
- `diagnostic_agent.py`: 調整診斷邏輯
- `scoring_system.py`: 改變評分標準

### Q4: 能用在其他設備嗎？
**A:**
可以！只需要：
1. 準備設備的感測器資料
2. 修改 `digital_twin.py` 的感測器定義
3. 更新故障類型

### Q5: Colab 版本怎麼用？
**A:**
1. 開啟 `notebooks/Training_System_Colab.ipynb`
2. 上傳到 Google Drive
3. 用 Colab 開啟
4. 執行所有 cell
