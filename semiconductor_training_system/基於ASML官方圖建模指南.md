## 🏗️ 建模策略（基於這張圖）

### 方案 A：精確還原（推薦！）

**優勢**：
- ✅ 有官方參考圖
- ✅ 內部結構清楚
- ✅ 零件位置精確
- ✅ 比例關係正確

**步驟**：

#### 第一步：在 Blender 中設定參考圖

```
1. 開啟 Blender
2. Add → Image → Reference
3. 載入你的 ASML 圖片
4. 調整圖片位置（側視圖）
5. 鎖定圖片（作為建模參考）
```

#### 第二步：建立外殼

```
根據圖片外輪廓：
1. 主體外殼（白色長方體）
   • 比例：約 3:4:5 (寬:高:深)

2. ASML Logo 區域（藍色）
   • 位置：正面左側

3. 控制面板（左側突出部分）
   • 螢幕、按鈕區域
```

#### 第三步：建立內部關鍵零件（對照圖片）

**零件 1：Projection optics（投影光學系統）**
```
位置：中間偏上
形狀：圓柱體組合（粉紅色高亮部分）
材質：金屬 + 局部透明玻璃
顏色：深灰色外殼，內部粉紅色發光
```

**零件 2：Wafer Stage（Wafer 載台）**
```
位置：底部中央
形狀：圓形平台
材質：深色金屬
特徵：可移動的載台結構
```

**零件 3：Reticle stage（Reticle 載台）**
```
位置：上方（Projection optics 上方）
形狀：方形載台
材質：金屬
特徵：光罩傳輸路徑
```

**零件 4：Wafer Handler（晶圓傳輸）**
```
位置：左側
形狀：機械手臂 + 傳送路徑
材質：金屬
特徵：晶圓進出口
```

**零件 5：UV - Level Sensor（感測器）**
```
位置：環繞 Projection optics
形狀：35 條光束（可用發光線條表示）
材質：Emission（發光）
顏色：淡紫色
```

#### 第四步：材質設定

**外殼**：
```
Material: Principled BSDF
Base Color: 白色 (0.95, 0.95, 0.95)
Metallic: 0.1
Roughness: 0.4
```

**藍色 ASML 區域**：
```
Base Color: ASML 藍 (0.2, 0.4, 0.8)
Metallic: 0.6
Roughness: 0.3
```

**粉紅色光學系統**：
```
Base Color: 粉紅色 (1.0, 0.4, 0.6)
Emission: 粉紅色 (強度 2.0)
Metallic: 0.8
Roughness: 0.2
```

**深色載台**：
```
Base Color: 深灰 (0.2, 0.2, 0.2)
Metallic: 0.9
Roughness: 0.3
```

#### 第五步：光源設定

```
1. 環境光：模擬無塵室照明
2. 內部發光：粉紅色光學系統
3. UV 感測光束：淡紫色線條（Emission）
```

## 🎨 詳細建模流程（跟著圖片做）

### Day 1：外殼結構（3 小時）

**參考圖片的外輪廓**：

```
1. 主體外殼
   • Add → Mesh → Cube
   • Scale: X=3, Y=5, Z=4（根據圖片比例）
   • 材質：白色

2. 藍色 ASML 區域
   • 選擇正面左側的面
   • Extrude（擠出）
   • 材質：ASML 藍

3. 控制面板突出
   • 左側添加小型 Cube
   • 添加螢幕和按鈕細節

4. ASML Logo
   • Add → Text
   • 轉換為 Mesh
   • 貼在藍色區域
```

### Day 2：內部光學系統（3 小時）

**參考圖片中粉紅色高亮部分**：

```
1. 主光學柱
   • Add → Cylinder
   • Height: 4
   • Radius: 1
   • 位置：中間偏上

2. 鏡頭組件
   • 在主柱上添加多個小圓柱（代表多層鏡片）
   • 材質：玻璃（Transmission: 0.5）

3. 粉紅色發光效果
   • 添加 Emission Shader
   • Color: 粉紅色
   • Strength: 2.0

4. 外殼包覆
   • 大圓柱包住光學系統
   • 材質：深灰金屬
```

### Day 3：載台系統（2 小時）

**參考圖片底部的 Wafer Stage**：

```
1. Wafer 載台
   • Add → Cylinder
   • Radius: 2
   • Height: 0.3
   • 位置：底部中央
   • 材質：深灰金屬

2. Wafer（晶圓）
   • 小圓盤放在載台上
   • 材質：銀灰色，有反光

3. Reticle 載台
   • Add → Cube
   • Scale 成薄平台
   • 位置：光學系統上方
   • 材質：金屬
```

### Day 4：傳輸系統與細節（2 小時）

**參考圖片左側的 Wafer Handler**：

```
1. 晶圓傳輸路徑
   • 左側添加機械手臂結構
   • 使用 Cylinder 組合

2. Pod 入口
   • 外殼左側開口
   • 添加門的細節

3. UV 感測器光束
   • Add → Curve → Bezier
   • 繞光學系統排列 35 條
   • Emission 材質（淡紫色）
```

### Day 5：材質與光源（2 小時）

```
1. 切換到 Shading 工作區
2. 為每個零件設定材質（參考上方設定）
3. 添加場景光源
4. 測試渲染效果
```

### Day 6：優化與導出（1 小時）

```
1. 檢查零件命名：
   • LensSystem
   • WaferStage
   • ReticleStage
   • CoolingSystem（可選）
   • VacuumChamber（可選）

2. 優化多邊形：
   • 使用 Decimate Modifier
   • 目標：< 100k polygons

3. 導出：
   • File → Export → glTF 2.0 (.glb)
   • Format: glTF Binary (.glb)
   • Include: All objects
   • Compression: 開啟
   • 儲存為：duv_machine.glb
```
