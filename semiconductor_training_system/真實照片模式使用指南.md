## 🚀 如何使用真實照片模式

### 步驟 1：準備 ASML DUV 設備照片

#### 方法 A：從 ASML 官網下載（推薦）

1. 訪問 ASML 官網：https://www.asml.com/en/products/duv-lithography-systems
2. 找到 TWINSCAN NXT 系列產品
3. 下載產品照片（通常在產品頁面有高解析度圖片）

#### 方法 B：Google 圖片搜尋

搜尋關鍵字：
```
ASML TWINSCAN NXT lithography
ASML DUV scanner
ASML 193nm lithography system
```

**建議規格**：
- 解析度：至少 1920x1080
- 格式：JPG 或 PNG
- 視角：正面或側面，能看清各部件

#### 方法 C：使用公司設備照片

如果你們公司有 ASML 設備，可以：
1. 拍攝實際設備照片
2. 確認可以用於內部訓練
3. 正面或 45 度角拍攝最佳

---

### 步驟 2：放置照片到指定位置

將下載的照片放到：

```
semiconductor_training_system/interface/images/asml_duv.jpg
```

或

```
semiconductor_training_system/interface/images/asml_duv.png
```

**使用終端機複製**：
```bash
# 假設你的照片在 Downloads 資料夾
cp ~/Downloads/asml_photo.jpg "c:\Users\user\Desktop\在職碩\OneDrive - 長庚大學\長庚碩班\論文\semiconductor_training_system\interface\images\asml_duv.jpg"
```

---

### 步驟 3：切換到真實照片模式

#### 方法 A：使用切換工具（最簡單）

```bash
cd "c:\Users\user\Desktop\在職碩\OneDrive - 長庚大學\長庚碩班\論文\semiconductor_training_system"
python 切換視覺化模式.py
```

然後選擇 `1` (真實照片模式)

#### 方法 B：手動修改（進階）

編輯 `interface/simulation_interface.py`：

找到這一行（大約在第 21 行）：
```python
from interface.equipment_visualizer_realistic import RealisticEquipmentVisualizer
```

替換為：
```python
from interface.equipment_visualizer_photo import PhotoEquipmentVisualizer
```

找到這一行（大約在第 50 行）：
```python
self.equipment_visualizer = RealisticEquipmentVisualizer()
```

替換為：
```python
self.equipment_visualizer = PhotoEquipmentVisualizer(
    use_local_image=True,
    image_path="interface/images/asml_duv.jpg"
)
```

---

### 步驟 4：啟動系統

```bash
python start_simulation.py
```

如果標記位置不準確，可以編輯 `equipment_visualizer_photo.py`：

```python
# 找到標記生成部分，調整座標
{self._generate_marker(

    height=40,
    label="曝光光源",
    ...

```

### 使用網路圖片（無需下載）

如果圖片在網路上：

```python
self.equipment_visualizer = PhotoEquipmentVisualizer(
    use_local_image=False,
    image_path="https://example.com/asml_duv.jpg"
```

## ❓ 常見問題

### Q1：找不到合適的 ASML 照片怎麼辦？

**A**：繼續使用**超真實 SVG 模式**，效果已經非常好了！
```bash
python 切換視覺化模式.py
# 選擇 2
```

### Q2：照片顯示不出來？

**A**：檢查：
1. 圖片路徑是否正確
2. 圖片格式是否支援（JPG/PNG）
3. 檔案名稱是否為 `asml_duv.jpg` 或 `asml_duv.png`

### Q3：可以用其他設備的照片嗎？

**A**：可以！只要是光刻機設備就行，例如：
- Nikon NSR 系列
- Canon FPA 系列
- 其他 ASML 型號

只需調整標記位置即可。

### Q4：標記位置不準確怎麼辦？

**A**：編輯 `equipment_visualizer_photo.py`，調整各標記的 `x` 和 `y` 座標。
