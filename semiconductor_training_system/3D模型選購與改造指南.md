## 📊 為什麼 TurboSquid 模型通常只有外殼？

### 原因分析

**1. 這是「視覺展示模型」，不是「工程模型」**

```
視覺展示模型 (Visual Model):
  • 用途：廣告、影片、遊戲
  • 重點：外觀好看
  • 內部：通常是空的
  • 價格：$50-$200

工程模型 (Engineering Model):
  • 用途：模擬、分析、維修訓練
  • 重點：結構精確
  • 內部：完整零件
  • 價格：$1000-$10000+（通常不公開販售）
```

**2. 內部結構是商業機密**

ASML 不會公開內部結構，所以 3D 藝術家只能根據外觀照片建模。

**3. 減少多邊形數量**

內部零件會大幅增加多邊形，影響載入速度。

---

## ✅ 解決方案：「外殼 + 關鍵零件」策略

### 核心概念

> 你不需要完整的內部結構，你只需要**可以「打開」看到關鍵零件的外殼**。

### 為什麼這樣就夠了？

**新人訓練的需求**：
1. ✅ 看到外觀（認識設備）
2. ✅ 看到關鍵零件位置（Wafer 載台、鏡頭、光源）
3. ✅ 看到異常告警位置（冷卻系統、真空腔體）
4. ❌ 不需要看到每個螺絲、每條管線

**數位孿生的需求**：
1. ✅ 3D 視覺化
2. ✅ 可以旋轉、縮放
3. ✅ 可以點擊零件查看資訊
4. ✅ 可以標示異常（紅燈）
5. ❌ 不需要物理精確的內部結構

## 🎨 Blender 快速添加內部零件教學

### 第一步：導入外殼模型

```
1. 開啟 Blender
2. File → Import → FBX/OBJ
3. 選擇下載的 ASML 模型
4. 確認模型正確顯示
```

### 第二步：創建內部零件

#### 零件 1：Wafer 載台

```
1. Add → Mesh → Cylinder
2. 參數設定：
   • Radius: 適當大小（根據外殼比例）
   • Depth: 0.2（薄平台）
3. 移動到底部中央
4. 材質：Metallic 金屬
5. 命名：WaferStage
```

#### 零件 2：鏡頭組

```
1. Add → Mesh → Cylinder
2. 參數設定：
   • Radius: 稍小
   • Depth: 4（長圓柱）
3. 移動到中間
4. 材質：Glass (Transmission: 0.8)
5. 命名：LensSystem
```

#### 零件 3：光源

```
1. Add → Mesh → Cube
2. 縮放到適當大小
3. 移動到頂部
4. 材質：
   • Emission 材質
   • Color: 淡藍色
   • Strength: 5.0
5. 命名：LightSource
```

#### 零件 4：冷卻管路（簡化版）

```
1. Add → Mesh → Cylinder
2. 縮放：細長型（管子）
3. 複製幾個，排列在側面
4. 材質：藍色金屬
5. 命名：CoolingSystem
```

#### 零件 5：真空腔體

```
1. Add → Mesh → Cylinder
2. 放大，包覆 Wafer 載台
3. 材質：
   • Glass
   • Transmission: 0.3（半透明）
4. 命名：VacuumChamber
```

### 第三步：調整材質

```
切換到 Shading 工作區：
1. 選擇外殼 → 確認是藍白配色
2. 選擇金屬零件 → Metallic: 0.9, Roughness: 0.3
3. 選擇玻璃零件 → Transmission: 0.8, Roughness: 0.05
4. 選擇光源 → Emission: 5.0
```

### 第四步：導出

```
1. File → Export → glTF 2.0 (.glb)
2. 設定：
   • Format: glTF Binary (.glb)
   • Include: All objects
   • Compression: 開啟
3. 檔名：duv_machine.glb
4. 儲存到：assets/models/
```

## 🎯 方案 B：使用「剖面視圖」(Cutaway View)

如果你只買到外殼模型，可以用「剖面」技巧：

### 概念

```
不是「打開」外殼，而是「切掉一半」：

正常視圖：
┌─────────────┐
│   ASML      │  ← 只看到外殼
│             │
└─────────────┘

剖面視圖：
┌──────
│ ASML ┊ ← 切掉右半邊
│  ●───┊ ← 可以看到內部零件
└──────
```

### Blender 實作

```
1. 選擇外殼模型
2. Edit Mode (Tab 鍵)
3. 選擇右半邊的所有面
4. Delete → Faces
5. 這樣就能看到內部了
```

### 在 Three.js 中切換

```javascript
// 添加切換按鈕
let showCutaway = false;

function toggleCutaway() {
    showCutaway = !showCutaway;

    if (showCutaway) {
        // 隱藏右半邊外殼
        shellRightHalf.visible = false;
        // 顯示內部零件
        internalComponents.visible = true;
    } else {
        shellRightHalf.visible = true;
        internalComponents.visible = false;
    }
}
```

## 🎬 方案 C：使用「透明外殼」

最簡單的方法：讓外殼半透明。

### Blender 設定

```
1. 選擇外殼模型
2. 材質設定：
   • Transmission: 0.5（半透明）
   • Alpha: 0.5
3. 這樣可以透過外殼看到內部零件
```

### Three.js 設定

```javascript
// 讓外殼半透明
shellMaterial.transparent = true;
shellMaterial.opacity = 0.3;

// 可以用滑桿調整透明度
opacitySlider.addEventListener('input', (e) => {
    shellMaterial.opacity = e.target.value;
});
```
