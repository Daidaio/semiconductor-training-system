## 📊 圖表1：核心流程圖

```
┌─────────────────────────────────────────────────────────────┐
│                     自適應教學系統流程                        │
└─────────────────────────────────────────────────────────────┘

    學員回答理論問題
           ↓
    ┌──────────────┐
    │ 多維度評估    │
    │ - 單次得分 30%│
    │ - 最近5次 40% │ ← 核心指標
    │ - 主題掌握 20%│
    │ - 學習趨勢 10%│
    └──────────────┘
           ↓
    ┌──────────────────────────────────────┐
    │      判斷理解程度與模式切換           │
    │                                      │
    │  平均≥8.5分 → 🎯 CHALLENGE (挑戰)   │
    │  平均6.5-8.5 → ✅ STANDARD (標準)   │
    │  平均4.0-6.5 → 🏗️ SCAFFOLDING (鷹架) │
    │  平均<4.0分  → 🔧 REMEDIAL (補救)   │
    └──────────────────────────────────────┘
           ↓
    ┌──────────────────────────────────────┐
    │        動態調整教學策略               │
    │                                      │
    │  問題難度：基礎 → 適中 → 進階         │
    │  反饋風格：詳細 → 平衡 → 簡潔         │
    │  提示強度：3級 → 1級 → 0級           │
    └──────────────────────────────────────┘
           ↓
    生成自適應問題 & 反饋
           ↓
    (循環：持續追蹤與調整)
```

## 🔄 圖表5：類比心理診斷AI

```
┌──────────────────────────────────────────────────────────┐
│                   心理診斷 AI                             │
└──────────────────────────────────────────────────────────┘

    病人回答憂鬱量表（PHQ-9）
           ↓
    ┌──────────────┐
    │ AI評估        │
    │ - 輕度 14-19分│ → 衛教建議
    │ - 中度 20-27分│ → 心理諮商
    │ - 重度 28+分  │ → 精神科轉介
    └──────────────┘
           ↓
    調整後續問診方式：
    - 輕度：開放式問題，鼓勵表達
    - 中度：結構化評估，深入探索
    - 重度：關懷語氣，評估自殺風險

════════════════════════════════════════════════════════════

┌──────────────────────────────────────────────────────────┐
│                 半導體訓練 AI（我們的系統）                │
└──────────────────────────────────────────────────────────┘

    學員回答理論問題
           ↓
    ┌──────────────┐
    │ AI評估        │
    │ - 優異 ≥8.5分 │ → 挑戰模式
    │ - 良好 6.5-8.5│ → 標準模式
    │ - 普通 4.0-6.5│ → 鷹架模式
    │ - 不佳 <4.0分 │ → 補救模式
    └──────────────┘
           ↓
    調整後續教學方式：
    - 優異：複雜問題，不給提示
    - 良好：適中難度，適度引導
    - 普通：分解步驟，頻繁提示
    - 不佳：基礎概念，大量提示


核心相似性：
  ✅ 都是根據「評估結果」動態調整「後續策略」
  ✅ 都有多個等級（不是簡單的好/壞二分法）
  ✅ 都是即時調整（不是事後才處理）
```

5. **價值論證**：用「圖表6」（傳統vs自適應）強調優勢
6. **成果展示**：用「圖表7」（學習分析報告）展示輸出
