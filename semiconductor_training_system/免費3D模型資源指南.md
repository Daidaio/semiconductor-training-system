
#### 搜尋：實驗室設備
```
https://sketchfab.com/search?q=laboratory+equipment&type=models&features=downloadable
```

#### 搜尋：製造設備
```
https://sketchfab.com/search?q=manufacturing+equipment&type=models&features=downloadable
```

#### 搜尋：科技機器
```
https://sketchfab.com/search?q=sci-fi+machine&type=models&features=downloadable
```

### 📋 如何使用 Sketchfab

**步驟 1：進入搜尋頁面**
- 點擊上方任一連結

**步驟 2：篩選免費可下載模型**
- 左側 Filters
- ✅ Downloadable: **Yes**
- ✅ Price: **Free** (點擊 "Free" 選項)

**步驟 3：查看模型**
- 點擊模型縮圖
- 可以在瀏覽器中 360° 旋轉預覽
- 查看多邊形數量（建議 < 100k）
- 查看授權類型

**步驟 4：下載模型**
- 點擊 **Download 3D Model** 按鈕
- 選擇格式：**glTF (.glb)** ✅ 最佳選擇
- 或選擇：**Original Format** → 用 Blender 轉換

**步驟 5：授權確認**
- 查看 License 標示
- ✅ **CC BY 4.0**：可用於商業，需標註作者
- ✅ **CC BY-SA 4.0**：可用於商業，需標註作者，衍生作品需相同授權
- ✅ **CC0 (Public Domain)**：完全自由使用
- ⚠️ **CC BY-NC**：僅限非商業使用

## 2. Free3D

### 🔗 直接搜尋連結

```
https://free3d.com/3d-models/industrial

https://free3d.com/3d-models/machine

https://free3d.com/3d-models/equipment
```

### 特點
- ✅ 完全免費
- ✅ 不需註冊即可下載
- ⚠️ 品質參差不齊
- ⚠️ 格式多為 .obj, .fbx（需要用 Blender 轉換）

### 下載步驟
1. 點擊模型
2. 點擊 **Download** 按鈕
3. 選擇格式（通常是 .obj 或 .fbx）
4. 用 Blender 轉換為 .glb

## 🎨 免費模型的優缺點

### ✅ 優點

1. **零成本**
   - 完全免費
   - 適合預算有限的學生

2. **快速驗證**
   - 可以先測試系統
   - 確認視覺效果可行性

3. **多樣選擇**
   - Sketchfab 有上萬個模型

### ❌ 缺點

1. **不一定找到真正的曝光機**
   - 需要用「類似」的設備替代
   - 可能需要解釋「這是概念模型」

2. **品質參差不齊**
   - 有些模型很簡陋
   - 有些模型材質缺失
   - 有些模型多邊形數過多

3. **可能需要修改**
   - 零件命名可能不符合
   - 需要用 Blender 調整
   - 材質可能需要重做
