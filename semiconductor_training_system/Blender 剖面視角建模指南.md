## 🚀 Step-by-Step 建模教學

### Phase 1：外框結構（30 分鐘）

#### Step 1：清空場景，保留參考圖

```
操作：
1. 選擇預設立方體（如果有的話）
2. 按 Delete 刪除
3. 保留你已載入的參考圖片
```

#### Step 2：建立主外框（藍色邊框部分）

```
操作：
1. 按 Shift+A → Mesh → Cube
2. 按 S 縮放，對照參考圖調整到差不多大小
3. 按 Tab 進入編輯模式
4. 按 3 切換到面選擇模式
5. 選擇「正面」的面（面向你的那面）
6. 按 Delete → 選擇 Faces（刪除正面，變成開口）
7. 按 Tab 回到物件模式
```

**結果**：一個沒有正面的盒子（可以看到內部）

#### Step 3：命名外框

```
操作：
1. 在右側 Outliner 找到物件
2. 雙擊改名為：OuterFrame
```

---

### Phase 2：粉紅色光學系統（45 分鐘）⭐最重要

這是整個模型的視覺焦點！

#### Step 4：建立光學系統主體

```
操作：
1. 按 Shift+A → Mesh → Cylinder
2. 在左下角彈出選單中：
   - Vertices: 32（圓滑度）
   - Radius: 1
   - Depth: 3（高度）
3. 按 R → X → 90（旋轉 90 度，讓圓柱直立對應光學系統方向）
4. 按 G 移動到參考圖中光學系統的位置
```

#### Step 5：設定粉紅色發光材質 ⭐

```
操作：
1. 選中光學系統圓柱
2. 到右側 Material Properties（紅色球圖示）
3. 點擊 + New 新增材質
4. 設定以下參數：

   Base Color: 點擊色塊
   ├─ R: 1.0 (255)
   ├─ G: 0.4 (100)
   └─ B: 0.6 (150)

   Emission: 打勾啟用
   ├─ Color: 同上粉紅色
   └─ Strength: 2.5（發光強度）

   Metallic: 0.3
   Roughness: 0.2
```

#### Step 6：複製多個光學元件

從參考圖可以看到光學系統有多層結構：

```
操作：
1. 選中光學系統圓柱
2. 按 Shift+D 複製
3. 按 Z 鎖定垂直移動
4. 移動到上方適當位置
5. 按 S 縮放調整大小
6. 重複 2-5 步驟，建立 3-4 層光學元件
```

**對照參考圖調整每層的大小和位置**

#### Step 7：命名光學系統

```
操作：
1. 選中所有光學元件（Shift+點擊）
2. 按 Ctrl+J 合併成一個物件
3. 改名為：LensSystem
```

---

### Phase 3：Wafer Stage 晶圓載台（30 分鐘）

#### Step 8：建立載台平台

```
操作：
1. 按 Shift+A → Mesh → Cylinder
2. 設定：
   - Radius: 2
   - Depth: 0.3（扁平的圓盤）
3. 移動到參考圖中 Wafer Stage 的位置（光學系統下方）
```

#### Step 9：設定深灰色金屬材質

```
操作：
1. 新增材質
2. 設定：
   Base Color: RGB(50, 50, 55) 深灰色
   Metallic: 0.9
   Roughness: 0.3
```

#### Step 10：命名

```
改名為：WaferStage
```

---

### Phase 4：Reticle Stage 光罩載台（20 分鐘）

#### Step 11：建立光罩載台

```
操作：
1. 按 Shift+A → Mesh → Cube
2. 按 S 縮放成扁平的方形
3. 移動到參考圖中光罩位置（光學系統上方）
```

#### Step 12：設定材質

```
操作：
1. 新增材質
2. Base Color: RGB(60, 60, 65)
3. Metallic: 0.8
4. Roughness: 0.3
```

#### Step 13：命名

```
改名為：ReticleStage
```

---

### Phase 5：連接管線（20 分鐘）

#### Step 14：建立管線

從參考圖可以看到有一些連接管線和線纜。

```
操作：
1. 按 Shift+A → Mesh → Cylinder
2. 設定：
   - Radius: 0.1（細管）
   - Depth: 2（長度）
3. 移動到適當位置
4. 複製多條，擺放在參考圖顯示的位置
```

#### Step 15：設定管線材質

```
操作：
1. 新增材質
2. Base Color: RGB(80, 80, 90) 灰色
3. Metallic: 0.7
4. Roughness: 0.4
```

#### Step 16：命名

```
改名為：CoolingPipes
```

---

### Phase 6：ASML Logo 面板（15 分鐘）

#### Step 17：建立 Logo 面板

```
操作：
1. 按 Shift+A → Mesh → Plane
2. 按 S 縮放到適當大小
3. 移動到參考圖中 ASML Logo 的位置（左下角）
4. 按 R → Y → 90 旋轉成垂直面板
```

#### Step 18：設定白色材質

```
操作：
1. 新增材質
2. Base Color: RGB(250, 250, 250) 白色
3. Metallic: 0.1
4. Roughness: 0.5
```

#### Step 19：命名

```
改名為：LogoPanel
```

## 💡 建模技巧

### 1. 隨時對照參考圖
```
快捷鍵：
- 數字鍵 1：正視圖
- 數字鍵 3：側視圖
- 數字鍵 7：俯視圖
- 數字鍵 0：相機視圖
```

### 2. 頻繁儲存
```
每完成一個元件就按 Ctrl+S
```

### 3. 善用複製
```
Shift+D 複製物件
比重新建立更快
```

### 4. 調整透明度看參考圖
```
選中參考圖片 → 右側 Object Data Properties
→ 調整 Opacity（透明度）讓它不要太擋視線
```

## 🚨 常見問題

### Q：物件看不到了？
**A**：可能被隱藏了。按 `Alt+H` 顯示所有隱藏物件。

### Q：顏色看起來不對？
**A**：切換到 Rendered 視圖模式（按 `Z` → 選擇 Rendered）

### Q：發光效果看不出來？
**A**：需要在 Rendered 模式下才看得到 Emission 效果。

### Q：如何精確輸入數值？
**A**：按 `G/S/R` 後直接輸入數字，例如 `G` `Z` `2` `Enter` = 往上移動 2 單位
