## 問題分析

### 原始問題
用戶點擊設備圖上的部件時沒有反應。

### 原因
Gradio 的 HTML 組件內的 JavaScript 無法直接訪問其他 Gradio 組件（隱藏的 textbox）。這是 Gradio 的安全限制。

### 解決方案
改用**實體按鈕**代替 JavaScript 點擊事件：
- 在設備圖下方添加 8 個部件選擇按鈕
- 使用 Gradio 原生的按鈕點擊事件
- 直接調用 Python 函數更新視圖

```python
def make_component_selector(component_name):
    """創建部件選擇函數"""
    def select_func(eq_html, dash_html, eq_status_html, conv_hist, act_log):
        return system.select_component(component_name, eq_html, dash_html, eq_status_html, conv_hist, act_log)
    return select_func

for btn, component_name in component_buttons:
    btn.click(
        fn=make_component_selector(component_name),
        inputs=[equipment_display, dashboard_display, equipment_status_display, system_messages, action_log],
        outputs=[equipment_display, dashboard_display, equipment_status_display, system_messages, action_log]
    )
```
