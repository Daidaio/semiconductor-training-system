## 圖表1: 總體判斷流程

```
┌─────────────────────────────────────────────────────────────┐
│                    學員回答理論問題                          │
│                    (例如：解釋熱膨脹)                        │
└────────────────────┬────────────────────────────────────────┘
                     │
                     ▼
        ┌────────────────────────┐
        │    AI 評估回答品質      │
        │                        │
        │  • 評分 0-10 分        │
        │  • 分析理解程度        │
        │  • 檢查概念正確性      │
        └────────────┬───────────┘
                     │
                     ▼
        ┌────────────────────────┐
        │   記錄到學習歷史        │
        │                        │
        │  recent_scores = [7,8,9]│
        └────────────┬───────────┘
                     │
                     ▼
        ┌────────────────────────────────┐
        │    計算最近 5 次平均分數        │
        │                                │
        │  avg = sum(recent_5) / 5      │
        │  例如：(7+8+9+8+9)/5 = 8.2    │
        └────────────┬───────────────────┘
                     │
                     ▼
        ┌────────────────────────────────┐
        │      判斷當前理解程度           │
        ├────────────────────────────────┤
        │  if avg >= 8.5:               │
        │    → excellent (卓越)          │
        │  elif avg >= 6.5:             │
        │    → good (良好)               │
        │  elif avg >= 4.0:             │
        │    → fair (尚可)               │
        │  else:                        │
        │    → poor (不佳)               │
        └────────────┬───────────────────┘
                     │
                     ▼
        ┌────────────────────────────────┐
        │      自動切換教學模式           │
        ├────────────────────────────────┤
        │  excellent → CHALLENGE 🎯     │
        │  good      → STANDARD ✅       │
        │  fair      → SCAFFOLDING 🏗️   │
        │  poor      → REMEDIAL 🔧      │
        └────────────┬───────────────────┘
                     │
                     ▼
        ┌────────────────────────────────┐
        │    調整教學策略參數             │
        ├────────────────────────────────┤
        │  • 問題難度                    │
        │  • 反饋詳細度                  │
        │  • 提示強度 (0-3級)            │
        │  • 問題類型                    │
        └────────────┬───────────────────┘
                     │
                     ▼
        ┌────────────────────────────────┐
        │     生成下一個問題              │
        │   (根據新的教學模式)            │
        └────────────────────────────────┘
```

## 圖表5: 學習軌跡範例

```
┌─────────────────────────────────────────────────────────────────┐
│          學員A的完整學習軌跡（7次問答）                          │
└─────────────────────────────────────────────────────────────────┘

得分
10 │                                         ●━━●
   │                                        ╱
 9 │                              ●━━━━━━━━╯    [CHALLENGE]
   │                             ╱                 ↑
 8 │                    ●━━━━━━━╯                 挑戰模式
   │                   ╱          [STANDARD]       複雜問題
 7 │          ●━━━━━━━╯                ↑          無提示
   │         ╱                         標準模式
 6 │   ●━━━━╯                          適中難度
   │       ╲
 5 │        ●                      [SCAFFOLDING]
   │         ↑                           ↑
 4 │         鷹架模式                    分解步驟
   │         分解引導                    多提示
 3 │
   │
 2 │
   │
 1 │
   │
 0 └───┬────┬────┬────┬────┬────┬────→ 問答次數
      Q1   Q2   Q3   Q4   Q5   Q6   Q7


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

詳細過程：

Q1: 「什麼是熱膨脹？」
   └─ 回答：「溫度升高時東西會變大」
      ├─ 得分：6分 (基本理解，但不深入)
      └─ 平均：6.0 → STANDARD 模式

Q2: 「熱膨脹會影響什麼？」
   └─ 回答：「嗯...可能會對不準？」
      ├─ 得分：5分 (概念模糊)
      ├─ 平均：(6+5)/2 = 5.5
      └─ 切換至 SCAFFOLDING 模式 ⚠️
         系統反應：分解步驟逐步引導

Q3: [鷹架引導後]
   └─ 回答：「膨脹程度不一樣，會產生相對移動」
      ├─ 得分：7分 (理解改善)
      ├─ 平均：(6+5+7)/3 = 6.0
      └─ 維持 SCAFFOLDING

Q4-Q5: 持續進步
   └─ 得分：8, 9分
      └─ 平均：(5+7+8+9)/4 = 7.25
         切換回 STANDARD ✅

Q6-Q7: 表現優異
   └─ 得分：9, 9分
      ├─ 最近5次：(7+8+9+9+9)/5 = 8.4
      └─ 接近 CHALLENGE (還差0.1分)

Q7 之後：
   └─ 平均達到 8.6 分
      └─ 切換至 CHALLENGE 模式 🎯
         系統給予複雜整合問題
```

  • 學員回答內容
  • 回答時間戳
  • 問題主題分類
        │
        ▼
   ┌──────────┐
   │ 評估引擎  │

        ▼

  • 單次得分 (0-10)

  • 學習趨勢 (improving/stable/declining)
        │
        ▼
