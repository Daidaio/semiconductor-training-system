## 📥 前置準備（15 分鐘）

### 1. 下載並安裝 Blender

**官方網站**：
```
https://www.blender.org/download/
```

**安裝步驟**：
1. 點擊 "Download Blender"
2. 選擇 Windows 版本（最新穩定版，目前是 4.x）
3. 下載完成後執行安裝檔
4. 安裝過程全部使用預設選項

**安裝完成後**：
- 桌面會有 Blender 圖示
- 雙擊開啟

### 2. 準備參考圖片

**將你的兩張 ASML 官方圖片存到桌面**：
```

  ├─ asml_side_view.jpg      (側視圖 - NXT:870B)
  └─ asml_top_view.jpg       (俯視圖 - NXT:2150i)
```

**圖片建議**：
- 裁切掉不需要的文字和邊框
- 保持圖片解析度適中（不要太小，不要太大）
- 另存新檔（不要修改原檔）

## 🚀 開始 Blender（第一次使用）

### Step 1：啟動 Blender

**雙擊 Blender 圖示**

你會看到：
```
┌─────────────────────────────────┐
│ Blender 4.x                     │
│                                 │
│   [歡迎畫面]                    │
│                                 │
│   □ General                     │
│   □ 2D Animation                │
│   □ Sculpting                   │
│   ...                           │
│                                 │
│   [ Get Started ]               │
└─────────────────────────────────┘
```

**操作**：
1. 不用選任何預設模板
2. 直接點擊畫面空白處或按 `Esc` 鍵關閉歡迎畫面
3. 你會看到預設場景（有一個立方體、燈光、相機）

### Step 2：認識 Blender 介面

```
┌────────────────────────────────────────────────────┐
│ [File] [Edit] [Render] ...          ← 頂部選單    │
├────────────────────────────────────────────────────┤
│ [🔧][⚙️][💡][📷][🎨]                  ← 工具列     │
├──────┬─────────────────────────────┬───────────────┤
│      │                             │               │
│ 左側  │      3D 視窗               │   右側        │
│ 工具  │   （這裡是建模區域）        │   屬性面板    │
│ 面板  │                             │               │
│      │        [立方體]             │               │
│      │                             │               │
├──────┴─────────────────────────────┴───────────────┤
│          Timeline（時間軸，建模時不用管）           │
└────────────────────────────────────────────────────┘
```

**重要區域**：
1. **3D 視窗（中央）**：你建模的地方
2. **工具列（左側）**：選擇、移動、旋轉等工具
3. **屬性面板（右側）**：設定物件屬性、材質

### Step 3：基本操作（必學）

**滑鼠操作**：
```
滑鼠中鍵按住拖曳    → 旋轉視角
滾輪上下滾動        → 縮放視角
Shift + 中鍵拖曳    → 平移視角
左鍵點擊           → 選擇物件
```

**鍵盤快捷鍵（超重要）**：
```
G 鍵 → Grab（移動物件）
  - 按 G 後移動滑鼠
  - 按 X/Y/Z 鎖定軸向
  - 按左鍵確認，右鍵取消

S 鍵 → Scale（縮放物件）
  - 按 S 後移動滑鼠
  - 按 X/Y/Z 鎖定軸向
  - 按左鍵確認

R 鍵 → Rotate（旋轉物件）
  - 按 R 後移動滑鼠
  - 按 X/Y/Z 鎖定軸向
  - 按左鍵確認

Tab 鍵 → 切換「物件模式」和「編輯模式」
Delete 鍵 → 刪除選中的物件
Ctrl+Z → 復原（撤銷）
```

**練習一下**（1 分鐘）：
1. 點擊立方體（變成橘色外框）
2. 按 `G` 鍵，移動滑鼠 → 立方體會跟著移動
3. 按滑鼠右鍵取消
4. 按 `S` 鍵，移動滑鼠 → 立方體會縮放
5. 按滑鼠右鍵取消
6. 按 `R` 鍵，移動滑鼠 → 立方體會旋轉
7. 按滑鼠右鍵取消

**如果誤操作了**：
- 按 `Ctrl+Z` 復原
- 或重開 Blender

## 🖼️ 載入參考圖片（30 分鐘）

### Step 4：清空預設場景

**目的**：刪除預設的立方體、燈光、相機，從乾淨場景開始。

**操作**：
1. 按住 `Shift`，依序點擊立方體、燈光、相機（全部選中，變成橘色）
2. 按 `Delete` 鍵
3. 出現選單，點擊 **Delete** 確認

**現在場景是空的**。

### Step 5：載入側視圖參考圖

**操作**：
1. 在上方選單點擊：**Add** → **Image** → **Reference**
2. 瀏覽到你的圖片：`桌面/ASML_Reference/asml_side_view.jpg`
3. 點擊 **Load Reference Image**

**你會看到**：
- 圖片出現在 3D 視窗中
- 圖片是「浮在空中」的平面

### Step 6：調整側視圖位置（對齊 Y 軸）

**目的**：讓側視圖垂直站立，方便對照建模。

**操作**：
1. 確保圖片被選中（橘色外框）
2. 按 `R` 鍵（旋轉）
3. 按 `X` 鍵（鎖定 X 軸）
4. 輸入 `90`（90 度）
5. 按 `Enter` 確認

**現在圖片應該垂直站立了**。

**調整圖片大小**（如果太大或太小）：
1. 按 `S` 鍵（縮放）
2. 移動滑鼠調整大小
3. 按左鍵確認

**移動圖片到側邊**（不要擋住建模區域）：
1. 按 `G` 鍵（移動）
2. 按 `X` 鍵（鎖定 X 軸）
3. 移動滑鼠，讓圖片往右側移動
4. 按左鍵確認

### Step 7：載入俯視圖參考圖

**操作**：
1. **Add** → **Image** → **Reference**
2. 選擇 `asml_top_view.jpg`
3. 點擊 **Load Reference Image**

**調整俯視圖位置**（保持水平）：
1. 圖片預設就是水平的，不需要旋轉
2. 按 `G` 鍵（移動）
3. 按 `Z` 鍵（鎖定 Z 軸）
4. 移動滑鼠，讓圖片往上移動（浮在空中）
5. 按左鍵確認

**現在你有兩張參考圖**：
- 側視圖（垂直）
- 俯視圖（水平）

### Step 8：鎖定參考圖（避免誤選）

**操作**：
1. 在右上角找到 **Outliner**（大綱視圖）
2. 找到兩張圖片的名稱（例如：`asml_side_view.jpg`）
3. 點擊圖片名稱右邊的 **箭頭圖示** → 禁用選擇

**現在參考圖不會被誤選了**。

## 🏗️ 建立主體外殼（1 小時）

### Step 9：新增立方體（主體基礎）

**操作**：
1. 按 `Shift+A`（開啟新增選單）
2. 選擇：**Mesh** → **Cube**
3. 立方體出現在 3D 視窗中心

### Step 10：調整立方體大小（對照參考圖）

**目的**：將立方體調整成 ASML 主體外殼的比例。

**操作**：
1. 確保立方體被選中
2. 按 `S` 鍵（縮放）
3. 按 `X` 鍵（鎖定 X 軸，寬度）
4. 輸入 `4`
5. 按 `Enter`

**這樣寬度變成 4 倍**。

**繼續調整深度**：
1. 按 `S` 鍵
2. 按 `Y` 鍵（鎖定 Y 軸，深度）
3. 輸入 `6`
4. 按 `Enter`

**繼續調整高度**：
1. 按 `S` 鍵
2. 按 `Z` 鍵（鎖定 Z 軸，高度）
3. 輸入 `4`
4. 按 `Enter`

**現在主體外殼的比例是 4×6×4（寬×深×高）**。

### Step 11：重新命名（方便管理）

**操作**：
1. 立方體被選中
2. 在右側屬性面板，找到最上方的物件名稱
3. 預設是 **Cube**
4. 雙擊改名為：**MainBody**
5. 按 `Enter` 確認

---

## 🎛️ 建立左側控制模組（30 分鐘）

### Step 12：新增立方體（左側模組）

**操作**：
1. 按 `Shift+A`
2. **Mesh** → **Cube**
3. 新立方體出現（可能和主體重疊）

### Step 13：調整左側模組大小

**操作**：
1. 按 `S`，`X`，輸入 `1.5`，`Enter`（寬度 1.5）
2. 按 `S`，`Y`，輸入 `2`，`Enter`（深度 2）
3. 按 `S`，`Z`，輸入 `4`，`Enter`（高度 4，和主體一樣高）

### Step 14：移動到左側

**操作**：
1. 按 `G` 鍵（移動）
2. 按 `X` 鍵（鎖定 X 軸）
3. 輸入 `-3`（往左移動 3 單位）
4. 按 `Enter`

**微調位置**（讓它貼在主體左側）：
1. 按 `G`，`X`，移動滑鼠
2. 觀察側視圖參考，調整到貼合
3. 按左鍵確認

### Step 15：命名左側模組

**操作**：
1. 在右側屬性面板，物件名稱改為：**LeftModule**

---

## 📦 建立右側儲存模組（30 分鐘）

### Step 16：複製左側模組（快速建立）

**操作**：
1. 選中 **LeftModule**
2. 按 `Shift+D`（複製）
3. 按 `X` 鍵（鎖定 X 軸）
4. 移動滑鼠往右
5. 輸入 `6`（移動到右側）
6. 按 `Enter`

### Step 17：調整右側模組高度（稍矮）

**操作**：
1. 右側模組被選中
2. 按 `S`，`Z`，輸入 `0.6`，`Enter`（高度變成 60%）

### Step 18：調整右側模組垂直位置（貼地）

**操作**：
1. 按 `G`，`Z`（移動 Z 軸）
2. 移動滑鼠，讓底部和主體底部對齊
3. 或輸入 `-0.8`（往下移動）
4. 按 `Enter`

### Step 19：命名右側模組

**操作**：
1. 物件名稱改為：**RightModule**

---

## 🎨 建立 ASML 藍色區域（45 分鐘）

### Step 20：選擇主體外殼

**操作**：
1. 點擊 **MainBody**
2. 按 `Tab` 鍵（進入編輯模式）

**你會看到**：
- 立方體的點、線、面都顯示出來了
- 上方工具列多了編輯工具

### Step 21：選擇正面的面（Front Face）

**操作**：
1. 在上方工具列，點擊 **面選擇模式**（方形圖示）
2. 旋轉視角，讓正面朝向你
3. 點擊正面的面（變成半透明藍色）

### Step 22：內縮（Inset）創建藍色區域

**操作**：
1. 按 `I` 鍵（Inset 工具）
2. 移動滑鼠往內縮
3. 縮到適當大小（約留 20% 邊距）
4. 按左鍵確認

**現在正面有一個「面中面」**。

### Step 23：擠出（Extrude）創建凹陷

**操作**：
1. 內縮的面保持選中
2. 按 `E` 鍵（Extrude）
3. 輸入 `-0.1`（往內凹陷 0.1 單位）
4. 按 `Enter`

### Step 24：回到物件模式

**操作**：
1. 按 `Tab` 鍵（退出編輯模式）

**現在主體正面有一個凹陷的區域，這會是 ASML 藍色區域**。

## 🎨 設定材質（30 分鐘）

### Step 25：切換到 Shading 工作區

**操作**：
1. 在 Blender 最上方，找到工作區標籤
2. 點擊 **Shading**

**你會看到**：
- 上半部：3D 視窗
- 下半部：Shader Editor（材質編輯器）

### Step 26：為主體設定白色材質

**操作**：
1. 選擇 **MainBody**
2. 在右側找到 **Material Properties**（紅色球形圖示）
3. 點擊 **+ New**（新增材質）

**你會看到**：
- 下方 Shader Editor 出現節點
- 預設是 **Principled BSDF** 節點

**設定白色**：
1. 在 **Principled BSDF** 節點中
2. 找到 **Base Color**（基礎顏色）
3. 點擊色塊，選擇白色 RGB(0.95, 0.95, 0.95)
4. **Metallic**：設定為 `0.1`
5. **Roughness**：設定為 `0.4`

### Step 27：為左側模組設定材質（複製白色）

**操作**：
1. 選擇 **LeftModule**
2. **Material Properties** → **+ New**
3. 複製上述白色材質設定

### Step 28：為右側模組設定材質

**操作**：
1. 選擇 **RightModule**
2. **Material Properties** → **+ New**
3. 複製白色材質設定

## 🔍 常見問題與解決

### Q1：找不到 Add → Image → Reference 選項？

**A1**：
- 確保你在 3D 視窗中（滑鼠在中央區域）
- 或使用快捷鍵 `Shift+A` 開啟選單

### Q2：物件移動時很卡頓？

**A2**：
- 降低參考圖片的解析度
- 或暫時隱藏參考圖（按 `H` 鍵）

### Q3：不小心刪除了重要物件？

**A3**：
- 立即按 `Ctrl+Z` 復原
- 或重新載入上次儲存的檔案：`File` → `Revert`

### Q4：比例看起來不對？

**A4**：
- 對照參考圖調整
- 選擇物件 → 按 `N` 鍵開啟屬性面板
- 在 **Dimensions** 中手動輸入尺寸

### Q5：視角被轉到奇怪的角度？

**A5**：
- 按數字鍵盤 `7`（俯視圖）
- 按數字鍵盤 `1`（正視圖）
- 按數字鍵盤 `3`（側視圖）
- 按數字鍵盤 `0`（相機視角）

## 💡 新手小技巧

1. **頻繁儲存**：`Ctrl+S` 每 10 分鐘按一次
2. **多用右鍵取消**：如果操作不滿意，右鍵取消，不要亂按左鍵確認
3. **Ctrl+Z 是好朋友**：不怕犯錯，隨時可以復原
4. **慢慢來**：第一次建模可能需要 5-6 小時，這很正常
5. **對照參考圖**：每完成一個步驟，都旋轉視角對照參考圖
