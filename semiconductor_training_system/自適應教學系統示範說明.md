# 自適應教學系統 - 根據回答品質動態調整教學策略

## 📋 系統概述

類似於**心理診斷 AI 根據病人回答調整診斷方式**，本系統根據學員的理解程度和回答品質，動態調整教學策略。

### 核心理念

```
心理診斷 AI：
病人回答 → 判斷憂鬱程度 → 調整問診方式（溫和/深入/轉介）

半導體訓練 AI：
學員回答 → 判斷理解程度 → 調整教學方式（挑戰/標準/輔導/補救）
```

## 🎯 判斷邏輯與調整策略

### 1. 多維度評估機制

#### 評估維度

| 維度 | 評估內容 | 權重 |
|------|----------|------|
| **單次得分** | 當前回答的理解程度 (0-10分) | 30% |
| **最近表現** | 最近5次的平均分數 | 40% |
| **主題掌握度** | 特定主題的歷史表現 | 20% |
| **學習趨勢** | 是否進步/退步/穩定 | 10% |

#### 理解程度分級

```python
分數範圍          理解程度              學員特徵
9-10分    →    excellent (卓越)    → 能舉一反三，理解透徹
7-8分     →    good (良好)         → 理解核心概念，表達清楚
5-6分     →    fair (尚可)         → 基本理解，但有遺漏
3-4分     →    poor (不佳)         → 部分理解，有明顯錯誤
0-2分     →    very_poor (很差)    → 完全不理解
```

---

### 2. 自適應教學模式

系統根據**最近5次平均分數**自動切換教學模式：

#### 模式 1: 挑戰模式 (Challenge) 🎯
**觸發條件**: 最近平均分 ≥ 8.5

**學員特徵**:
- 連續高分表現
- 對基礎概念掌握紮實
- 需要更高挑戰

**教學調整**:
```python
問題難度: advanced (跨領域整合)
反饋風格: concise (簡潔專業)
提示強度: 0 (不給提示)
反問問題: 複雜情境，需要整合多個概念
```

**範例對話**:
```
學員回答: [得分 9分] "溫度上升會導致熱膨脹，進而影響對準精度，
          同時光學折射率改變也會降低曝光品質。"

系統反饋: "✅ 你已經掌握了溫度對多個子系統的影響。"

系統反問: [挑戰模式]
"假設在一個 CVD 製程中，溫度突然從 350°C 上升到 420°C，
同時真空度從 1e-6 Torr 降至 5e-6 Torr。
請分析這兩個異常如何相互影響，並預測可能的連鎖故障。"
```

---

#### 模式 2: 標準模式 (Standard) ✅
**觸發條件**: 最近平均分 6.5 ~ 8.5

**學員特徵**:
- 表現穩定良好
- 基本概念理解正確
- 適合正常進度

**教學調整**:
```python
問題難度: moderate (適中)
反饋風格: balanced (平衡詳細度)
提示強度: 1 (適度提示)
反問問題: 檢驗核心概念，連結實際應用
```

**範例對話**:
```
學員回答: [得分 7分] "溫度上升會影響設備的精度。"

系統反饋: "✅ 正確！溫度確實會影響精度。
          你可以進一步思考：具體是哪些機制導致精度下降？"

系統反問: [標準模式]
"溫度上升時，曝光機的對準系統會受到什麼影響？
提示：想想熱膨脹對機械結構的作用。"
```

---

#### 模式 3: 鷹架模式 (Scaffolding) 🏗️
**觸發條件**: 最近平均分 4.0 ~ 6.5

**學員特徵**:
- 概念理解不完整
- 需要逐步引導
- 容易遺漏關鍵點

**教學調整**:
```python
問題難度: basic (基礎)
反饋風格: detailed (詳細解釋)
提示強度: 2 (較多提示，分解步驟)
反問問題: 分解成小步驟，逐步引導推理
```

**範例對話**:
```
學員回答: [得分 5分] "溫度太高會有問題。"

系統反饋: "[鷹架引導] 你的方向是對的！溫度確實會造成問題。
          讓我們一步步來理解：
          1️⃣ 溫度上升時，金屬和材料會發生什麼變化？（提示：體積）
          2️⃣ 這個變化會如何影響精密設備的機械結構？
          3️⃣ 最終會導致什麼製程問題？"

系統反問: [鷹架模式]
"我們先從第一步開始：
當溫度從 23°C 升高到 50°C 時，金屬零件的體積會變大還是變小？
這個現象叫什麼？"
```

---

#### 模式 4: 補救模式 (Remedial) 🔧
**觸發條件**: 最近平均分 < 4.0

**學員特徵**:
- 基礎概念不穩固
- 連續答錯或答非所問
- 需要重新建立基礎

**教學調整**:
```python
問題難度: fundamental (最基礎)
反饋風格: very_detailed (非常詳細 + 鼓勵)
提示強度: 3 (大量提示，幾乎手把手)
反問問題: 是非題或簡單選擇題，確保能答對
```

**範例對話**:
```
學員回答: [得分 2分] "不知道，可能是設備壞了。"

系統反饋: "[基礎鞏固] 沒關係，這個概念確實不容易。
          讓我先解釋一下熱膨脹的基本定義：

          熱膨脹 = 材料受熱時體積會變大

          日常例子：鐵軌在夏天會膨脹，所以軌道間要留縫隙。
          半導體例子：晶圓在加熱時直徑會變大約 0.1mm。

          💡 關鍵：溫度↑ → 體積↑ → 尺寸改變 → 精度下降"

系統反問: [補救模式 - 是非題]
"請問：當溫度上升時，金屬零件會膨脹變大嗎？
A. 會
B. 不會

（提示：想想鐵軌的例子）"
```

## 🧠 知識追蹤系統

### 主題分類與追蹤

系統自動將問題分類到 6 個核心主題：

```python
主題分類：
├─ thermal (熱學)：熱膨脹、溫度、冷卻、散熱
├─ vacuum (真空)：真空度、壓力、泵浦、洩漏
├─ optical (光學)：對準、曝光、光強、折射
├─ mechanical (機械)：定位、夾持、振動
├─ chemical (化學)：CVD、蝕刻、沉積、清洗
└─ electrical (電氣)：電源、電壓、電流、接地
```

### 主題表現追蹤

```python
每個主題記錄：
- avg_score: 平均得分
- attempts: 測試次數
- trend: 趨勢 (improving/stable/declining)
- mastery_level: 掌握程度 (mastered/proficient/developing/struggling)
```

### 知識盲點識別

```python
自動識別弱點主題：
if 主題平均分 < 5.0:
    標記為「需要加強」
    → 系統會在後續訓練中針對性提問
```

## 📊 完整運作流程示範

### 情境：學員 A 的完整學習軌跡

#### 第 1 次問答
```
Q: "什麼是熱膨脹？"
A: [系統解釋]
學員回答: "就是溫度升高時，東西會變大。"

評分: 6分 (fair - 基本理解)
主題: thermal
模式: STANDARD (初始模式)

反饋: "✅ 基本概念正確！可以進一步思考：
      什麼東西會變大？為什麼會變大？"

反問: [標準難度]
"在半導體製程中，溫度上升時晶圓和腔體都會膨脹。
這可能導致什麼問題？"
```

#### 第 2 次問答
```
學員回答: "嗯...可能會對不準？"

評分: 5分 (fair - 概念模糊)
平均分: (6+5)/2 = 5.5
模式: 切換至 SCAFFOLDING (分解引導)

反饋: "[鷹架引導] 方向正確！讓我們一步步分析：
      1️⃣ 晶圓和腔體膨脹係數不同
      2️⃣ 導致相對位移
      3️⃣ 影響對準精度"

反問: [鷹架模式 - 分解步驟]
"我們先確認：膨脹係數是什麼意思？
它代表溫度每升高1°C，材料長度變化的比例。
請問：如果兩種材料的膨脹係數不同，加熱時會發生什麼？"
```

#### 第 3 次問答
```
學員回答: "膨脹程度不一樣，會產生相對移動。"

評分: 7分 (good - 理解改善)
平均分: (6+5+7)/3 = 6.0
模式: 維持 SCAFFOLDING

反饋: "✅ 很好！你已經理解核心概念了。"

反問: [逐漸增加難度]
"那你覺得如果要減少熱膨脹的影響，可以採取什麼設計措施？"
```

#### 第 4-5 次問答
```
學員持續答對，得分 8, 9

平均分: (6+5+7+8+9)/5 = 7.0
模式: 切換至 STANDARD
```

#### 第 6 次問答
```
學員得分: 9分

最近5次平均: (7+8+9+8+9)/5 = 8.2
模式: 即將切換至 CHALLENGE (再一次高分就切換)

反問: [標準但偏難]
"優秀！接下來試試這個：
如果 CVD 製程中溫度控制失效，從 350°C 振盪到 400°C，
你會如何快速判斷是控制系統問題還是加熱器硬體故障？"
```

#### 第 7 次問答
```
學員得分: 9分

最近5次平均: (8+9+8+9+9)/5 = 8.6 ≥ 8.5
模式: 🎯 切換至 CHALLENGE

反饋: "[挑戰模式] 你已經掌握基礎概念，接下來挑戰整合應用。"

反問: [挑戰模式 - 複雜情境]
"情境：曝光機在 wafer 曝光過程中，發現以下異常同時發生：
1) 溫度從 23°C 上升到 28°C
2) 對準誤差從 5nm 增加到 45nm
3) 光學強度下降 3%
4) 冷卻水流量正常

請推理：
a) 這些異常之間的因果關係？
b) 最可能的根本原因是什麼？
c) 你會如何驗證你的推論？

（不提供提示，請獨立分析）"
```

## 🔄 動態調整的關鍵機制

### 1. 連續表現追蹤
```python
recent_scores = [最近5次得分]

if avg(recent_scores) >= 8.5:
    mode = CHALLENGE
elif avg(recent_scores) >= 6.5:
    mode = STANDARD
elif avg(recent_scores) >= 4.0:
    mode = SCAFFOLDING
else:
    mode = REMEDIAL
```

### 2. 主題弱點針對性訓練
```python
if 主題 in weak_topics:
    # 該主題多提問
    # 降低該主題的問題難度
    # 提供額外的參考資料連結
```

### 3. 學習趨勢預警
```python
if 最近3次得分 < 前3次得分 - 2:
    ⚠️ 警告: "表現下滑，建議休息或降低難度"

if 最近3次得分 > 前3次得分 + 2:
    ✅ 鼓勵: "進步明顯！可以挑戰更難的主題"
```

## 💡 類比：心理診斷 AI

### 心理診斷 AI 的邏輯
```
病人回答憂鬱量表
  ↓
AI 判斷憂鬱程度：
  - 輕度 (14-19分) → 衛教建議
  - 中度 (20-27分) → 心理諮商
  - 重度 (28+分)  → 立即轉介精神科
  ↓
後續問診方式調整：
  - 輕度：開放式問題，鼓勵表達
  - 中度：結構化評估，深入探索
  - 重度：關懷語氣，評估自殺風險
```

### 半導體訓練 AI 的對應邏輯
```
學員回答理論問題
  ↓
AI 判斷理解程度：
  - 卓越 (9-10分) → 挑戰模式
  - 良好 (7-8分)  → 標準模式
  - 尚可 (5-6分)  → 鷹架模式
  - 不佳 (<5分)   → 補救模式
  ↓
後續教學方式調整：
  - 挑戰：複雜情境，不給提示
  - 標準：適中難度，適度引導
  - 鷹架：分解步驟，逐步引導
  - 補救：基礎概念，大量提示
```

## 🚀 系統優勢

1. **個人化學習路徑**
   - 根據每個學員的表現動態調整
   - 避免「一刀切」教學

2. **即時適應調整**
   - 不需要等到課程結束才調整
   - 每次問答後立即評估並調整

3. **知識盲點識別**
   - 自動追蹤弱點主題
   - 針對性加強訓練

4. **學習動機維持**
   - 能力強的學員：給挑戰，避免無聊
   - 能力弱的學員：降難度，避免挫折
   - 進步中的學員：適度鼓勵，維持動力

5. **數據驅動教學**
   - 完整記錄學習軌跡
   - 提供可視化的進步曲線
   - 支援教學研究分析
