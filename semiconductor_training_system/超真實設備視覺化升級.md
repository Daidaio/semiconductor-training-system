## 🌟 照片級真實感的實現技巧

### 技巧 1：多層光影
```
設備主體
  ↓
外層陰影（feGaussianBlur + feOffset）
  ↓
金屬反光（radialGradient）
  ↓
內層凹陷（innerDepth filter）
  ↓
最終效果：立體感十足
```

### 技巧 2：材質分層
```
鋁外殼（5 層漸層）
  + 高光點（白色 opacity 0.8）
  + 陰影面（深灰 opacity 0.5）
  = 金屬質感
```

### 技巧 3：玻璃效果
```
半透明底色（淺藍）
  + 放射狀漸層（中心亮，邊緣暗）
  + 模糊濾鏡（feGaussianBlur）
  = 真空腔體玻璃
```

### 技巧 4：細節堆疊
```
設備主體
  + 螺絲（4 個角落）
  + 格柵（11 條線）
  + 標籤（銘牌）
  + 指示燈（LED）
  = 工業設備感
```
