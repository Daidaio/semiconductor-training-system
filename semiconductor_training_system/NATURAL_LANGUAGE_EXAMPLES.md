## ✅ 支援的口語表達方式

#### 正式說法：
```
檢查冷卻水流量
查看溫度
確認真空系統
```

```

✅ 看看過濾網
✅ 冷卻水情況如何
✅ 溫度多少
✅ 查一下真空
```

### 2. **詢問專家/學長**

### 3. **執行動作**

#### 調整參數
```
正式： 調整冷卻水流量到 5.5

      ✅ 流量調一下
      ✅ 改一下溫度
      ✅ 弄一下流量
```

#### 更換零件
```
正式： 更換過濾網
口語： ✅ 換過濾網
      ✅ 換掉過濾網
      ✅ 換個新的濾網
      ✅ 裝新的過濾器
```

#### 清潔設備
```
正式： 清潔光學鏡片
口語： ✅ 清一下鏡片
      ✅ 擦鏡片
      ✅ 洗一下光學元件
      ✅ 清理鏡頭
```

#### 停機操作
```
正式： 停機
      緊急停機
口語： ✅ 先停一下
      ✅ 趕快停
      ✅ 停下來
      ✅ 關掉
      ✅ 緊急停止
```

#### 校正對準
```
正式： 執行對準校正
口語： ✅ 校一下對準
      ✅ 對一下
      ✅ 校正位置
```

## 🎯 完整對話範例

### 範例 1：發現冷卻問題

```
你： 溫度怎麼樣
系統： 鏡頭溫度：24.8°C（正常：22-24°C）
      溫度趨勢：[上升中] ↑↑↑

你： 學長，為什麼溫度一直上升
學長： 嗯，溫度上升通常和散熱有關。你覺得冷卻系統的哪個地方可能有問題？

你： 看一下冷卻水
系統： [檢查冷卻水系統]
      冷卻水流量計顯示：3.5 L/min（正常範圍：4.5-5.5）
      管路壓力：2.3 bar
      [警告] 流量偏低，可能影響散熱效果

你： 流量這麼低，該怎麼辦
學長： 流量異常通常是過濾器的問題。看看過濾網的狀況？

你： 看看過濾網
系統： [檢查過濾網]
      過濾網狀態：堵塞
      上次更換時間：3個月前
      [警告] 過濾網已堵塞，建議立即更換

你： 先停一下
系統： [執行停機程序]
      1. 通知產線主管... ✓
      2. 保存當前製程數據... ✓
      ...
      ✓ 安全停機完成，可以進行維護作業

你： 換過濾網
系統： [更換filter]
      1. 關閉冷卻水閥門... ✓
      2. 卸下舊過濾網... ✓
      3. 安裝新過濾網... ✓
      ...
      ✓ 更換完成！冷卻水流量已恢復正常。

學長： 很好，處理流程很標準。更換過濾網後記得觀察一下溫度有沒有降下來。
```

## 📝 支援的口語用詞表

### 動詞（動作）
| 正式 | 口語 |
|-----|------|
| 檢查 | 看、看一下、看看、瞧瞧、查、查一下 |
| 調整 | 調、調一下、改、改一下、弄、弄一下 |
| 更換 | 換、換掉、換新的、裝新的 |
| 清潔 | 清、擦、洗、清一下、擦一下 |
| 停機 | 停、關、停下來、關掉、先停 |
| 詢問 | 問、請問、問一下、問問 |

### 疑問詞
| 用法 | 範例 |
|-----|------|
| ...怎麼樣 | 冷卻水怎麼樣、溫度怎麼樣 |
| ...正常嗎 | 流量正常嗎、真空正常嗎 |
| ...有問題嗎 | 光源有問題嗎 |
| ...OK嗎 | 參數OK嗎 |
| 為什麼... | 為什麼溫度上升 |
| 該怎麼... | 該怎麼辦、該怎麼處理 |

### 稱呼
- 學長（最自然！）
- 專家
- 問一下
- 請問

## 💡 使用技巧

### ✅ 好的習慣：

1. **用疑問句檢查**
   ```
   ✅ 冷卻水正常嗎
   ✅ 溫度怎麼樣
   ✅ 流量OK嗎
   ```

2. **用口語化動詞**
   ```
   ✅ 看一下溫度
   ✅ 調一下流量
   ✅ 換過濾網
   ```

3. **自然地問學長**
   ```
   ✅ 學長，為什麼會這樣
   ✅ 問一下該怎麼辦
   ✅ 有什麼建議嗎
   ```

### ❌ 要避免：

1. **過於簡短無法理解**
   ```
   ❌ 看
   ❌ 怎麼辦
   ❌ 為什麼
   ```
   → 要說明看什麼、什麼該怎麼辦

2. **完全沒有關鍵字**
   ```
   ❌ 這個東西不太對
   ❌ 好像有點怪怪的
   ```
   → 要說明是哪個系統或參數

3. **過於複雜的句子**
   ```
   ❌ 我覺得可能是冷卻系統的過濾網或許因為長時間使用而導致堵塞所以才會...
   ```
   → 簡單直接就好
